
# 戰國三代記系統概覽 (v1.4)

> **v1.4 更新說明**：補充評議系統、武將評價系統、戰後處置系統；統一武將屬性為四面向；忍者任務統一為六種。

## 1. 遊戲主架構
- **核心系統**：進入方式、命令操作、政務邏輯
- **封建等級體系**：大名 → 宿老／家老 → 部將／侍大將 → 組頭

## 2. 領國系統
- **領地三層級**：令制國、城、郡
- **領地分配與賞賜邏輯**
- **郡屬性類型**：
  - 農業郡
  - 商業郡
  - 軍事郡
- **基礎設施與升級規則**

## 3. 家臣團系統
- **本家家臣團與一、二級家臣團結構**
- **屬臣編制與兵力上限**
- **指揮鏈與晉升封閉邏輯**
- **家族系譜分類**：一門、譜代、外樣

## 4. 封建權謀系統
- **忠誠度與野心**、派系與內部競爭
- **家臣晉升與牽制**、叛變風險
- **家臣互動**：推舉、聯名、構陷等
- **勢力內部治理與信賴度動態**

## 5. 軍事系統 (v1.2)
- **軍事行動類型**：
  - 攻郡
  - 攻城包圍
  - 強攻
- **軍力來源**：城兵、農兵
- **武將能力四面向**（詳見 0011.武將評價模組）：
  - 軍事（攻擊力、防禦力、士氣效率）
  - 智謀（計謀成功率、外交、特務）
  - 政治（內政治理、軍糧消耗控制）
  - 魅力（人望、說服、士氣恢復）
- **軍資與士氣機制**
- **指令系統與主從命令邏輯**
- **城池防禦層級**：本丸、一之丸、二之丸等

## 6. 情報系統 (v1.0)
- **情報類型**：軍力、國政、部署、忠誠等
- **忍者任務（六種，以 0008.情報模組為準）**：
  - 刺探
  - 暗殺
  - 籠絡
  - 離間
  - 造謠
  - 破壞
- **成功率、風險與效果**
- **忍者屬性與行動時程**

## 7. 評議系統 (v1.0)
- **定位**：遊戲內重大決策的核心議事機制
- **評議類別**：行政評議、外交評議、軍事評議、特別評議
- **觸發方式**：玩家主動召開 或 特定行動自動觸發（詳見 0007.評議模組）
- **影響**：家臣忠誠度變化、派系勢力消長、事件伏筆

## 8. 武將評價系統 (v1.0)
- **定位**：以文字描述取代傳統數值顯示武將能力
- **四面向評價**：軍事 / 智謀 / 政治 / 魅力，各分 E→S 六級
- **評價隨行動演化**，詳見 0011.武將評價模組

## 9. 戰後處置系統 (v3.0)
- **觸發條件**：佔領敵方領地後
- **決策流程**：領地處置 → 家臣團處理 → 不滿度系統 → 統治恢復
- **詳見** 0009.戰後處置模組

## 10. 玩家流程與操作
- **進入選項**：
  - 新遊戲
  - 讀取
  - 自創武將
  - 稱號一覽
- **常駐命令列**：存讀、情報、回主畫面等
- **政務操作**：
  - 官職任命
  - 郡治理
  - 法令
  - 視察
  - 典禮
