## 四、NPC AI 行為規格（MVP 最小版本）

### 4.1 設計原則

> MVP 的 AI 不追求智慧，只追求**不呆滯**。讓玩家感受到對手有在行動即可。

### 4.2 決策邏輯（每回合執行一次）

```
【今川家 AI 決策流程】

Step 1：評估兵力比
  攻守比 = 今川家總兵力 / 織田家總兵力

Step 2：決定行動傾向
  攻守比 ≥ 1.5 → 70% 機率選擇「攻擊行動」
  攻守比 1.0~1.5 → 40% 機率選擇「攻擊行動」
  攻守比 < 1.0 → 10% 機率選擇「攻擊行動」（以守為主）

Step 3a：若選擇「攻擊行動」
  - 選擇兵力最多的一級家臣團出征
  - 攻擊目標：與今川領地鄰接的織田領地中，守軍最少的郡
  - 採圍城（若目標為城）或野戰（若目標為郡）

Step 3b：若選擇「發展行動」
  - 隨機選擇 1 個己方郡
  - 若金錢 ≥ 200 且有可升級設施 → 執行升級
  - 否則 → 跳過（本回合不行動）
```

### 4.3 MVP AI 限制（刻意簡化）

| 限制項目 | 說明 |
|---------|------|
| 不使用農兵 | 農兵集結邏輯過複雜，NPC 僅用城兵 |
| 不進行外交 | 外交系統不在 MVP 範圍 |
| 不隨機事件觸發 | 事件系統 MVP 不實作 |
| 無資源危機處理 | NPC 軍糧耗盡時直接停止軍事行動 |
| 無繼承與派系 | 不計算忠誠度變化 |

### 4.4 NPC 資源維持規則

為避免 AI 因資源耗盡而完全停止活動（造成無聊遊戲體驗）：

- 今川家每月固定獲得基礎收入（不受設施影響）：**+150 金、+100 軍糧**
- 此為 MVP 暫時機制，正式版改為完整資源計算

## 六、MVP 靜音版聲音設定

**MVP 階段採「靜音版」**，無背景音樂、無音效。  
UI 操作回饋以視覺效果（高亮、動畫）代替音效。

> 詳細音效/音樂規格見未來版本文件（已登記於工作日誌待辦）。
