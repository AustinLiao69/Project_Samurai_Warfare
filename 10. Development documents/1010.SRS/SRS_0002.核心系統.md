## 1. 引言

### 1.1 目的
本文件定義《戰國三代記》核心系統（CoreSystem）的完整軟體需求，作為程式設計師實作的依據，並作為 SA 驗收的基準。

### 1.2 範圍
核心系統是遊戲的入口與骨架，負責：
- 遊戲啟動與場景初始化
- 回合流程管理
- 常駐命令列（Command Bar）
- 政務主選單（9 大政務與子選單）
- 地圖系統（三層級顯示模式）
- 存檔 / 讀檔系統

本 SRS **不涵蓋**各子系統的內部邏輯（家臣團、領國、軍事等），僅定義核心系統對這些模組的呼叫介面。

### 1.3 術語定義

| 術語 | 說明 |
|------|------|
| 回合 | 遊戲時間單位，每回合代表一個月 |
| 政務模式 | 每回合玩家可執行行政指令的階段 |
| 常駐命令列 | 畫面下方始終可見的快捷操作列 |
| 令制國 / 城 / 郡 | 領地三層級，由大到小 |

- `0000.Product_SPEC.md` v1.1

## 2. 系統概述

### 2.1 模組定位
CoreSystem 是 MainGame 場景的根節點，所有其他系統節點均掛載其下或由其初始化。

```
MainGame (Node)
└── CoreSystem (Node)
    ├── GameStateManager
    ├── SaveSystem
    ├── UIManager
    │   ├── CommandBar
    │   └── PoliticsMenu
    ├── MapSystem
    └── [其他子系統 by reference]
```

### 2.2 外部相依模組

| 相依模組 | 相依方式 | 用途 |
|---------|---------|------|
| RetainerSystem | 呼叫 | 政務：任命官職、家臣管理 |
| TerritorySystem | 呼叫 | 政務：治理郡城、授予知行 |
| CouncilSystem | 呼叫 | 政務：評定會議 |
| MilitarySystem | 呼叫 | 政務：召集農民兵、出兵指令 |
| IntelligenceSystem | 呼叫 | 常駐命令列：情報查看 |
| SaveSystem | 呼叫 | 常駐命令列：存讀檔 |

### 2.3 回合流程（文字狀態機）

```
[回合開始]
  → 事件結算（EventSystem）
  → 收入結算（TerritorySystem.collect_all）
  → [政務階段] 玩家操作政務選單
  → [軍事階段] 玩家下達軍令（可選）
  → [評議觸發檢查] CouncilSystem.check_auto_trigger
  → [情報階段] IntelligenceSystem.process_missions
  → 回合結束自動存檔
  → [下一回合]
```

## 3. 功能需求（SA 視角）

### 3.1 遊戲入口（主選單）

**FR-CORE-001**：系統啟動時顯示主選單，提供下列四個選項：

| 選項 | 功能 |
|------|------|
| 從頭開始 | 以預設劇本初始化遊戲資料並進入地圖畫面 |
| 讀取存檔 | 顯示存檔槽列表（最多 10 槽），選擇後載入遊戲 |
| 自創武將 | 開啟武將創建介面，允許設定姓名、初始評價 |
| 稱號一覽 | 開啟唯讀畫面，顯示玩家已解鎖的稱號列表 |

**FR-CORE-002**：「從頭開始」須呼叫 `SaveSystem.init_new_game()` 並以 PM008 初始資料為基礎初始化 `GameData`。

### 3.2 常駐命令列（CommandBar）

**FR-CORE-003**：遊戲進行中，畫面下方常駐命令列提供下列 8 項指令，任何時候（政務/軍事階段）均可使用：

| 編號 | 指令 | 行為 |
|------|------|------|
| 1 | 讀取存檔 | 呼叫 SaveSystem.load_game，需確認對話框 |
| 2 | 儲存存檔 | 呼叫 SaveSystem.save_game，需選擇槽位 |
| 3 | 回到主畫面 | 確認後返回主選單場景 |
| 4 | 勢力情報 | 開啟勢力資訊面板（只讀） |
| 5 | 朝廷情報 | 開啟朝廷狀態面板（只讀） |
| 6 | 幕府情報 | 開啟幕府狀態面板（只讀） |
| 7 | 武將情報 | 開啟家臣列表面板（呼叫 RetainerSystem.get_all） |
| 8 | 地圖切換 | 循環切換地圖顯示模式（見 FR-CORE-007） |

### 3.3 政務主選單

**FR-CORE-004**：政務階段，玩家可從第一層選單選擇 9 項政務。每回合政務次數上限待平衡設計確認（預設：無上限，MVP 可先不限制）。

| 編號 | 政務 | 呼叫目標 |
|------|------|---------|
| 1 | 任命官職 | RetainerSystem.assign_role() |
| 2 | 治理郡城 | TerritorySystem.govern_district() |
| 3 | 施行法令 | CoreSystem.apply_law() |
| 4 | 視察地方 | CoreSystem.inspect_territory() |
| 5 | 舉行典禮 | CoreSystem.hold_ceremony() |
| 6 | 評定會議 | CouncilSystem.open_council() |
| 7 | 召集農民兵 | MilitarySystem.recruit_peasants() |
| 8 | 家臣管理 | RetainerSystem.manage_retainers() |
| 9 | 授予知行 | TerritorySystem.grant_territory() |

**FR-CORE-005**：各政務項目的第二層子選單：

- **典禮**：圍獵 / 茶會 / 慶賀 / 接見使節（各有不同忠誠度效果）
- **視察地方**：進入勢力圖點選郡城，觸發隨機事件（農民邀請、陳情、賞賜請求、遇匪）
- **評定會議**：政務提案 / 軍事計畫 / 人事任免 / 情報行動
- **召集農民兵**：進入勢力圖選擇郡城，顯示可動員兵力、代價、農業影響
- **家臣管理**：任免家臣 / 評定忠誠 / 獎賞家臣（家寶 / 感謝狀 / 賜字 / 改姓）
- **授予知行**：選封地（郡 / 城）→ 指定受封者 → 確認知行分配

### 3.4 地圖系統

**FR-CORE-006**：地圖採「戰情室風格」，呈現方式為攤開在桌上的古風地圖（紙質/布質材質），以立體旗幟、小雕像與繩索路徑標示動態。

**FR-CORE-007**：地圖支援三種顯示模式，可循環切換：

| 模式 | 顯示內容 |
|------|---------|
| 勢力圖 | 各勢力以家紋旗幟與顏色標示領地 |
| 外交形勢圖 | 紅＝敵對、藍＝同盟、綠＝婚姻、紫＝從屬 |
| 分封圖 | 家臣團領地分配，以不同旗幟符號表示 |

**FR-CORE-008**：點擊令制國 → 進入地方地圖模式（城郡層級），可查看郡的設施、產能、駐兵、忠誠度。

**FR-CORE-009**：地圖上顯示己方軍勢（立體軍旗：主將名、兵力、行軍方向）。若未偵查敵方，敵軍以模糊謠言標示；刺探成功後顯示完整情報。

### 3.5 存檔 / 讀檔系統

**FR-CORE-010**：存檔槽位最多 10 個，每槽顯示：存檔名稱、遊戲日期（年月）、勢力名稱、縮圖。

**FR-CORE-011**：每回合結束自動存檔至 Auto Slot（不占用玩家槽位）。

**FR-CORE-012**：存檔格式使用 Godot Resource（`.tres`），序列化 `GameData`，大小目標 < 10MB。

| 項目 | 需求 |
|------|------|

## 5. 資料模型

```
GameData (Resource)
├── current_turn: int              # 回合數（月）
├── current_year: int              # 遊戲年份
├── player_faction_id: String      # 玩家勢力 ID
├── factions: Array[Faction]
├── retainers: Array[Retainer]
├── territories: Array[Territory]
└── game_flags: Dictionary         # 劇情旗標
```

## 6. GDScript 介面規格（PG 視角）

### 6.1 列舉型別

```gdscript
enum GamePhase {
    MAIN_MENU,
    POLITICS,
    MILITARY,
    COUNCIL,
    INTELLIGENCE,
    TURN_END
}

enum MapMode {
    FACTION,      # 勢力圖
    DIPLOMACY,    # 外交形勢圖
    TERRITORY     # 分封圖
}

enum CeremonyType {
    HUNTING,      # 圍獵
    TEA,          # 茶會
    CELEBRATION,  # 慶賀
    AUDIENCE      # 接見使節
}

enum RewardType {
    TREASURE,     # 家寶
    COMMENDATION, # 感謝狀
    NAME_GRANT,   # 賜字
    SURNAME_GRANT # 改姓
}
```

### 6.2 核心系統節點

```gdscript
class_name CoreSystem extends Node

# --- 狀態變數 ---
var current_phase: GamePhase = GamePhase.MAIN_MENU
var current_map_mode: MapMode = MapMode.FACTION
var game_data: GameData

# --- 子系統參考（由場景注入）---
@onready var save_system: SaveSystem
@onready var ui_manager: UIManager
@onready var retainer_system: RetainerSystem
@onready var territory_system: TerritorySystem
@onready var council_system: CouncilSystem
@onready var military_system: MilitarySystem
@onready var intelligence_system: IntelligenceSystem

# --- 回合管理 ---
func start_new_turn() -> void
func end_turn() -> void
func get_current_turn() -> int
func get_current_year_month() -> Dictionary  # {"year": int, "month": int}

# --- 政務行動 ---
func apply_law(law_type: String, params: Dictionary) -> Dictionary
    # 回傳: {"success": bool, "loyalty_changes": Dictionary, "message": String}

func inspect_territory(territory_id: String) -> Dictionary
    # 回傳: {"event_type": String, "event_data": Dictionary}

func hold_ceremony(ceremony_type: CeremonyType) -> Dictionary
    # 回傳: {"gold_cost": int, "loyalty_changes": Dictionary}

func reward_retainer(retainer_id: String, reward_type: RewardType) -> bool

# --- 地圖系統 ---
func switch_map_mode() -> MapMode   # 循環切換，回傳新模式
func enter_local_map(territory_id: String) -> void
func exit_local_map() -> void

# --- 存檔系統（委派 SaveSystem）---
func save_game(slot: int) -> bool
func load_game(slot: int) -> bool
func get_save_slot_info(slot: int) -> Dictionary
    # 回傳: {"name": String, "year": int, "month": int, "faction": String, "thumbnail": Texture2D}
```

### 6.3 存檔系統節點

```gdscript
class_name SaveSystem extends Node

const SAVE_DIR: String = "user://saves/"
const AUTO_SLOT: int = 0
const MAX_SLOTS: int = 10

func init_new_game(scenario_id: String) -> GameData
func save_game(slot: int, data: GameData) -> bool
func load_game(slot: int) -> GameData   # null 表示讀取失敗
func auto_save(data: GameData) -> bool
func list_slots() -> Array[Dictionary]  # 所有槽位資訊
func delete_slot(slot: int) -> bool
```

### 6.4 訊號（Signals）

```gdscript
# CoreSystem 訊號
signal turn_started(turn: int)
signal turn_ended(turn: int)
signal phase_changed(old_phase: GamePhase, new_phase: GamePhase)
signal map_mode_changed(new_mode: MapMode)
signal game_saved(slot: int)
signal game_loaded(slot: int)
signal event_triggered(event_type: String, event_data: Dictionary)
```
