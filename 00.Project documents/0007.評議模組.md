# 戰國三代記 — 評議系統設計 v1.0

## 系統定位
- 評議系統是遊戲內所有重大決策的核心機制。
- 所有政策（行政、外交、軍事、內務）必須透過評議，由大名與重臣、領主共同討論。
- 評議結果影響：
  - 領主與重臣的 **滿意度**
  - 派系勢力平衡
  - 事件觸發機率（叛變、支持、結盟等）

---

## 1. 評議觸發條件

### 1.1 評議類別
- **行政評議**：開墾農田、修築城牆、獎勵商業、徵收稅賦
- **外交評議**：聯姻、結盟、宣戰、和議、上洛
- **軍事評議**：出兵攻打、調兵遣將、防禦布署
- **特別評議**：繼承人任命、領地分封、重大法令

### 1.2 召集對象
- 直屬家老
- 分封領主（依地位、資歷）
- 特殊職位（軍奉行、政務奉行、奉行衆等）

---

## 2. 評議流程

### 2.1 召開評議
- 玩家（大名）選擇議題
- 系統生成參與者名單

### 2.2 參與者表態
- 每位與會者根據：
  - 個人利益（封地、稅收、軍權）
  - 派系立場（本家、外樣、反對派）
  - 性格特質（保守、進取、謹慎、冒險）
- 系統生成文字意見：宿老‧山內盛員：「當前糧草不足，恐難長期作戰，望殿下三思。」（反對）
家老‧高坂昌信：「若不先取北信濃，將來恐腹背受敵，臣願為先鋒。」（贊成）

- 表態結果：
- 贊成 / 反對 / 中立
- 附理由敘述

### 2.3 大名裁決
- 大名有最終決定權：
- **多數贊成 → 執行容易**
- **多數反對但仍強行執行 → 滿意度下降**
- **多數贊成但被否決 → 贊成派滿意度下降**

### 2.4 會後影響
- **滿意度變化**：基於立場一致或衝突而調整
- **派系勢力變動**：支持/反對的派系力量增減
- **事件伏筆**：滿意度過低可能觸發拒命、叛變、秘密結盟等事件

---

## 3. 滿意度變化機制

| 狀況 | 滿意度變化範圍 |
|------|---------------|
| 立場與裁決一致 | +5 ～ +15 |
| 立場與裁決相反 | -5 ～ -20 |
| 獲得封賞、官職 | +10 ～ +25 |
| 地位被削弱 | -15 ～ -30 |
| 連續多次被忽視 | 額外 -10 ～ -20 |

- 滿意度以文字提示：高坂昌信眉宇舒展，顯然對殿下的決策十分滿意。北條綱成面色陰沉，沉默不語。

---

## 4. 派系與影響
- 每位參與者屬於某一派系（本家派、譜代、外樣、反對派等）
- 派系立場影響評議表態傾向
- 大名偏袒某派系會造成另一派系集體不滿
- 派系勢力會隨滿意度變化而增減，影響未來評議結果

---

## 5. 玩家體驗目標
- 每一次評議都是政治博弈，而不只是單純的選項
- 玩家必須在政策效率與派系平衡之間取捨
- 長期不顧派系情緒會帶來重大內亂風險
