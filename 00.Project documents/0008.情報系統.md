
# 情報系統

## 一、情報類型設計（可被收集與揭露的資訊）

| 分類 | 子項目 | 特殊說明 |
|------|--------|----------|
| 外部情報 | 國力、軍力、內政等級（農、商、軍）、金糧存量 | 越強的勢力越難刺探 |
| 外交情報 | 與他國的關係、盟約、潛在敵意 | 有時候也能從造謠或離間中反推 |
| 軍事部署 | 各城兵力、將領配置、部隊移動 | 高階忍者才能刺探 |
| 家臣團情報 | 家臣忠誠、野心、政治立場（親主/中立/親敵） | 可暴露叛變危險或潛在同盟 |
| 城內情報 | 城池設施、防衛強度、資源儲量 | 可供破壞或奪取目標判斷 |
| 群眾情緒 | 民心士氣、是否受敵國影響、是否遭間諜鼓動 | 造謠、離間的依據 |

## 二、忍者系統設計

| 元素 | 說明 |
|------|------|
| 等級 | 決定行動成功率、可執行任務範圍（如高級忍者才可暗殺大名） |
| 技能專精 | 可偏重刺探、破壞、操縱（籠絡/離間）、暗殺等，不同任務有不同加成 |
| 行動時長 | 根據距離、對象、行動類型不同，所需天數不同，行動中不可用 |
| 成功率 | 受敵國警戒度、防諜等級、自家忍者等級與技能影響 |
| 身分暴露風險 | 失敗可能導致身分暴露，引起外交或國內信任問題 |
| 派遣機制 | 可派多名忍者合力行動，或進行分工（如一人造謠、一人破壞） |

## 三、情報與行動結果機制

| 行動類型 | 成功效果 | 失敗懲罰 | 特殊備註 |
|---------|----------|----------|----------|
| 刺探 | 獲得目標國特定情報 | 無收穫／忍者被捕／外交惡化 | 資訊有機率為假 |
| 暗殺 | 對方將領死亡／重傷 | 忍者被捕／失敗無損 | 高風險高報酬 |
| 籠絡 | 提升目標家臣忠誠／轉為己用 | 忍者身分暴露／家臣反感 | 須長期培養信任 |
| 離間 | 降低目標家臣忠誠／挑起內鬥 | 忍者被察覺／目標忠誠提升 | 民心低易成功 |
| 造謠 | 降低目標民心／引起動亂 | 被識破造成我方聲譽損害 | 可配合軍事行動干擾敵人 |
| 破壞 | 損壞設施、斷補給、降士氣 | 忍者被捕／敵人提高戒備 | 損敵不須戰 |

## 額外補強建議

### 情報等級系統：
每次刺探只能獲得一層情報，如「軍力評估 → 將領名單 → 部隊位置 → 移動路線」，需多次行動。

### 防諜系統：
可派武將設置防諜網，或訓練反間部隊提升守城安全。

### 情報真假機制：
可設"誤導情報"，玩家可能被假情報誤導而作出錯誤決策。

### 內政忍者部署：
可設「情報府」機構養忍者，也可向他國雇用「浪人忍者」執行特定高風險任務。
