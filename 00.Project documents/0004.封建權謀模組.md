
# 封建權謀系統 v1.0

## 一、設計理念

遊戲偏向封建社會結構，強調「家臣團」的分層統治、內部矛盾與權力鬥爭。風格接近《權力遊戲》那樣的鬥爭與背叛，玩家不僅經營勢力，也需應對派系分裂、謀反、擁立等局勢。

## 二、家臣團體系回顧

### 家臣團等級
- **本家家臣團**：大名直屬，由宿老／家老領導
- **一級家臣團**：宿老或家老統轄，擁有自身的屬臣
- **二級家臣團**：由部將／侍大將領導
- **三階層指揮結構**，形成可獨立發展與對抗的大型家臣團體

## 三、權謀系統設計

### 1. 家臣忠誠度與立場
- 每位家臣有「忠誠度」與「政治立場」屬性
- 忠誠度非數值顯示，而是**文字分級**
- 初始忠誠度會隨時間、事件、職位、獎賞與恐嚇變動

| 忠誠分級 | 描述 | 內部忠誠範圍 | 情報獲取方式 |
|---------|------|-------------|-------------|
| 臣服如山 | 絕對忠誠 | 91–100 | 初始或忍者探查成功 |
| 忠心耿耿 | 忠誠穩定 | 76–90 | 忍者探查成功 |
| 表面恭敬 | 外表順從 | 61–75 | 忍者探查成功 |
| 意圖未明 | 搖擺不定 | 41–60 | 未探查或情報過期 |
| 暗懷異志 | 潛藏異心 | 21–40 | 高階忍者探查成功 |
| 潛在叛徒 | 策動叛亂中 | 0–20 | 高階忍者連續探查成功 |

### 2. 忍者與情報系統
- 忍者不僅能探查敵勢力，也能監控己方家臣
- 若**未派遣忍者探查家臣**，則忠誠度可能停留在過期資訊
- 忍者等級影響情報準確性與獲得機率

### 3. 家臣派系與內鬥
- 家臣可組成不同擁立派系，支持不同繼承人選（嫡子、養子、分家等）
- 大名子孫元服或收養養子後，會自動誘發擁立分裂情勢
- 派系間可能：
  - **爭奪繼承權**
  - **策反他人家臣團**
  - **暗殺主君或其他家臣**
  - **支持他門擁立新主君**

### 4. 分裂與叛亂事件
- 一級與二級家臣團之間亦可產生權力鬥爭
- 敵對派系可能鼓動二級家臣團背叛其上級，導致**家臣團內戰或分裂出走**

## 四、情報系統延伸
- 情報更新需要持續派遣忍者，否則家臣立場將保持過期狀態
- 若誤判家臣立場，可能導致：
  - 無預警的謀反或背叛
  - 被敵派系滲透失去要地
  - 錯誤任命導致派系失衡

## 五、未來擴充方向（草案）

- 家臣間互動事件（賄賂、聯姻、仇殺）
- 情報偽造／誤導可能性設計
- 忠誠動態可視化系統
- 擁立戰爭與分家內戰演算系統
