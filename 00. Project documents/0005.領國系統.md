
# 領國系統 v1.1

## 結構與層級：

### 令制國：
- **層級**：最高層級
- **子類型**：包含多座城，且可新建城
- **功能**：大名可以設定所有城與郡的總體政策

### 城：
- **層級**：第二層級，城是統治的中心
- **子類型**：可設有多個郡
- **功能**：每座城可下轄若干郡，並可設定各城的年貢比例及相關策略

### 郡：
- **層級**：第三層級，為基本行政單位
- **子類型**：
  - 農業郡：提供金錢和農業資源
  - 商業郡：提供金錢和商業資源
  - 軍事郡：提供軍糧和軍事資源
- **資源**：每個郡會有不同的資源（農業郡、商業郡、軍事郡分別提供金錢、軍糧等）

## 年貢與忠誠度系統：

### 年貢設置：
大名可以為每個郡設定年貢比例（例如10%～50%），或者設定固定的年貢金額（例如500金、200軍糧），根據郡的資源產量來決定具體數字。

### 忠誠度影響：
- 當年貢比例設定過高時，會導致家臣團的忠誠度下降，這會使該郡更容易發生叛亂
- 忠誠度與年貢比例成反比，若年貢超過郡的資源產量上限，會導致資源枯竭
- **忠誠度變化公式**：忠誠度下降 = (年貢比例 - 目標年貢比例) × 影響因子

### 叛亂風險：
- 若年貢過高且持續時間過長，郡的叛亂風險會顯著提高
- 當忠誠度降至某個臨界值時，叛亂風險將達到100%，並可能觸發該郡的叛亂
- **叛亂風險公式**：叛亂風險 = (忠誠度低下時的年貢比例 / 高年貢設定的比例) × 時間

### 策略建議：
- 大名應該根據郡的狀況來調整年貢比例，若某些郡資源豐富，可以適當提高年貢比例，反之，則應減少年貢要求以避免叛亂

### 影響範圍：
- 高年貢會降低郡的基礎設施升級速度，並影響農業、商業等設施的運作
- 這也會使得家臣團忠誠度降低，增加內部不穩定的風險

## 設施升級系統：

每個郡最多可以有五個基礎設施，並且分為農業、商業、軍事三大類。每類設施會有三個升級層級，並且升級消耗金錢。

### 農業設施：
**村落 → 升級為 農村 → 升級為 農莊**

**升級成本**：
- 村落 → 農村：100金，1月
- 農村 → 農莊：300金，2月
- 總升級時間：3月

**升級所需金錢**：
- 村落：100金
- 農村：300金
- 農莊：700金

### 商業設施：
**市集 → 升級為 商街 → 升級為 行會**

**升級成本**：
- 市集 → 商街：120金，1月
- 商街 → 行會：360金，2月
- 總升級時間：3月

**升級所需金錢**：
- 市集：120金
- 商街：360金
- 行會：800金

### 軍事設施：
**駐屯地 → 升級為 兵營 → 升級為 軍陣營**

**升級成本**：
- 駐屯地 → 兵營：150金，1月
- 兵營 → 軍陣營：400金，2月
- 總升級時間：3月

**升級所需金錢**：
- 駐屯地：150金
- 兵營：400金
- 軍陣營：900金

## 資源管理與軍糧管理：

### 資源：
軍糧與金錢是兩大關鍵資源。

### 郡的年貢：
主要以金錢進行升級，並且由大名指派對郡的資源徵收。

### 軍糧與金錢的消耗：
每次軍事行動、設施升級等都會消耗相應的金錢或軍糧。
