
# 人物立繪生成指引

請遵照下列指示生成日本戰國武將圖片

## 一、風格定位：「內政型戰國奉行官」立繪設計

**風格取向**：本圖明顯採用了類似《信長之野望・新生》或《大志》系列中「譜代家臣」或「奉行官僚」類角色的立繪風格。

**職能象徵**：角色並非軍將型的披甲上陣形象，而是以佩刀文官姿態呈現，強調行政、財政或奉行管治能力。

**表情設計**：冷峻、沉穩，顯示其作為權威奉行的慎密與堅定，眼神銳利卻不張揚，帶有一種「代官型」智將氣質。

## 二、服裝與道具細節分析

| 元素 | 描述 | 象徵意義 |
|------|------|----------|
| 烏帽子 | 採用高立黑色烏帽子 | 表示其官職或文人身份，非戰場武士裝束 |
| 深色狩衣風和服 | 棕黑基調，胸配雙家紋 | 代表古河足利氏麾下奉公眾的正式服制，內敛卻不失威嚴 |
| 家紋圖案 | 三階紋（可能是丸に九曜或類似圖案） | 暗示其家族或仕官對象，有別於將門家系的獨立性 |
| 佩刀位置 | 腰際橫佩，手握膝上 | 呈現身為武士身分的象徵，然而其握法非備戰，而是平時作息之態 |

## 三、構圖風格：歷史模擬遊戲的典型角色立繪結構

**視角構圖**：採正面坐姿、半身構圖，極似《信長之野望》歷代版本立繪角色設定圖。

**背景處理**：仿古色暈染背景，略帶煙霧質感，聚焦人物本身，無需場景，凸顯其為「人事系內政角」。

**肌理與質感**：衣物有明顯布質摺痕與質感層次，臉部紋理與陰影刻畫細緻，具寫實風格與威儀感。

## 四、角色印象與定位建議（可應用於遊戲）

| 遊戲系統 | 建議定位 |
|----------|----------|
| 《信長之野望・新生》 | 【政務官】【家老】【郡代】【直轄城代】等強化內政角色 |
| 能力值設計建議 | 統率：52、武勇：38、智略：78、政務：91、忠誠：95 |
| 特性技能 | 「訴願奉行」（提升安定度）、「代官任命」（農業效率↑）、「内政の達人」 |

## 五、總評

這張畫像有效轉譯了原始「古河奉行眾」古畫風人物形象，將其內斂沉穩的氣質，成功轉化為具有遊戲角色魅力的內政型武士官僚立繪。在《信長之野望》的世界觀中，他會是那種「不登戰場卻掌握一地興亡」的關鍵人物——如江戶幕府的勘定奉行或古河城下的實務主官。
