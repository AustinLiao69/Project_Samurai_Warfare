# 日本戰國武將角色設計（僧兵型）

## 一、風格定位：「武裝宗教勢力代表・僧兵」立繪設計

- **風格取向**：本圖風格融合《信長之野望・新生》系列中延曆寺、石山本願寺等宗教勢力下的「僧兵」或「宗門代官」形象，兼具宗教威嚴與軍事氣勢。
- **職能象徵**：此角色雖披僧衣，卻為實際領導武裝僧侶的戰鬥指揮官，具備對外抗戰與內部統治雙重職能。
- **表情設計**：面容剃淨、神情嚴肅，眼神不怒自威，流露出宗教信仰下的剛烈與使命感，表現出「誓死護法」的信念。

## 二、服裝與道具細節分析

| 元素             | 描述                                                  | 象徵意義                                               |
|------------------|-------------------------------------------------------|--------------------------------------------------------|
| 剃髮／僧頭       | 全頭剃淨，無髮飾物                                    | 表明其僧侶身份，為出家之人，與武士造型明顯區分         |
| 黑紅法衣         | 外披深紅或墨黑法衣，腰束繩結，繡有佛教宗派徽紋        | 結合宗教與軍事象徵，傳達其武僧領袖的宗門權威形象       |
| 僧兵鎧片         | 法衣內可見部分小札鎧甲護胸                            | 顯示其身兼僧侶與戰士身份，具備戰鬥與保衛寺社之功能       |
| 武器道具         | 長柄薙刀立於身旁，或手握佛珠與木魚                    | 薙刀象徵僧兵武裝，佛具則代表信仰與法力                   |
| 背飾旗幟         | 背後隱見小型幡幟，書有「南無阿彌陀佛」或「不動明王」文字 | 表明其護法戰僧身份，並彰顯宗教領袖的正統與神聖         |

## 三、構圖風格：宗教武力勢力的典型立繪風格

- **視角構圖**：正面坐姿，半身至膝構圖，雙手持佛珠或握武器，姿態穩重如山。
- **背景處理**：以沉黑與赭紅為基調，背景隱見寺廟、火焰或法器輪廓，營造出濃厚宗教武士氛圍。
- **肌理與質感**：法衣布料明顯摺紋，鎧片細節精緻，臉部線條嚴正而不失悲憫感，具寫實而神聖的宗教力量美學。

## 五、總評

此角色設計巧妙融合了宗教與軍事形象，展現了日本戰國時期「僧兵」的雙重特質 —— 既是信仰堅定的佛門弟子，亦是抗戰一方的悍勇武者。在《信長之野望》類遊戲中，此類角色可作為中立勢力的宗門防衛首領、或本願寺系統中的主要戰將，具有高忠誠、高士氣激勵效果的戰略價值。他是那種「不求天下，卻能護一方清淨」的存在，是武與佛合一的戰國異數。
