# 日本戰國武將角色設計（忍者型）

## 一、風格定位：「情報內政型隱密役・忍者」立繪設計

- **風格取向**：本圖風格接近《信長之野望・新生》系列中甲賀、伊賀、風魔等忍者角色，非單純戰場角色，而是更具「情報＋內政」雙重職能的幕後要角。
- **職能象徵**：角色造型非典型軍將，亦非文官傳統衣著，而是介於隱密行動與謀略活動間的忍者裝束，突顯其「政略間諜」角色屬性。
- **表情設計**：神情內斂、眼神銳利，嘴角含笑但不親和，透露出一種難以揣測的氣質——彷彿早已洞悉一切棋局布局。

## 二、服裝與道具細節分析

| 元素             | 描述                                                   | 象徵意義                                              |
|------------------|--------------------------------------------------------|-------------------------------------------------------|
| 頭巾／頭罩       | 深灰包頭忍者頭巾，遮額露眼，無髮飾                     | 表現其伊賀流或甲賀流忍者傳統，強調潛伏與識別性         |
| 深色裝束         | 緊身黑灰色忍裝，肩部有簡易布鎧                         | 強調機動與潛行能力，非正規軍裝，便於暗行行動            |
| 家紋圖案         | 某隱密宗家或密使機構之小型刺繡紋章                     | 指涉其幕後效忠勢力（如北條、今川等），非公開身份        |
| 佩刀與暗器       | 腰佩短打刀，另藏苦無或飛鏢於袖內                       | 象徵「即使文職亦可致命」，其非戰場武將卻具決定性手段    |
| 道具細節         | 腰掛信件筒、小卷軸、火摺與墨包                          | 突顯情報收集、潛入、破壞與傳令等任務實務性道具配置      |

## 三、構圖風格：內政型忍者角色立繪構圖

- **視角構圖**：半身坐姿，雙手交疊於膝上，身形微側，似時時準備起身行動。
- **背景處理**：昏暗陰影交錯，背景模糊處理，仿似夜色或秘境，與其他正規家臣角色有所區隔。
- **肌理與質感**：衣物具緊實布料細節，具磨損與折痕，臉部明暗反差強烈，突顯忍者特有的冷靜與敏捷。

## 五、總評

這名忍者型角色成功融合了傳統忍者的行動力與內政系統中情報收集的關鍵職能。其非前線武將，卻足以在戰國亂世中左右勢力興亡，擁有潛入、滲透、策反、調查等多元能力。在《信長之野望》系列中，他可作為敵後干擾型的關鍵角色，亦可內政強化中擔任「情報官」角色——可謂「看不見的第三勢力」，在陰影中重塑光明的版圖。
