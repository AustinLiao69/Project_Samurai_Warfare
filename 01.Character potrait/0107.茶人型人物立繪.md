# 日本戰國武將角色設計（茶人型）

## 一、風格定位：「文化內政型茶道奉行」立繪設計

- **風格取向**：本圖延續《信長之野望・新生》或《大志》系列中偏重文化與儀式職能的家臣立繪風格，呈現出一位深諳茶道與政略平衡的文化智將。
- **職能象徵**：角色非軍武之將，亦非單純文官，而是以「茶道」為核心，兼具外交、藝術、精神統御功能的內政文化型角色。
- **表情設計**：神情安詳內斂，眉目間透出一絲不動如山的沉穩，眼神如靜水流深，體現茶道「和敬清寂」的氣韻與威儀。

## 二、服裝與道具細節分析

| 元素           | 描述                                                   | 象徵意義                                          |
|----------------|--------------------------------------------------------|---------------------------------------------------|
| 烏帽子變化款     | 採用簡化版黑布烏帽，略有文士風格修飾                     | 保留官職象徵，又融入茶人之雅士氣質                     |
| 素雅茶人裝束     | 淡墨色無家紋小袖與羽織，選用自然布料，講究摺痕與布理         | 反映茶道的「侘寂」美學與反奢之道                        |
| 手持茶道具       | 手持茶杓或茶碗，膝上放置帛巾與茶筅                         | 彰顯其茶人身份，象徵文化權威與精神儀式之掌控者            |
| 刀劍象徵配件     | 腰際佩短刀，未作備戰姿態，刀柄包有布繩裝飾                 | 象徵其身分為武士階層之一，但更重精神與文化掌控             |
| 茶印徽紋         | 使用代表茶道流派的家紋（如利休家或千家風紋）                 | 象徵仕奉關係與文化派系，非傳統武家紋章                    |

## 三、構圖風格：文化官僚型茶人角色立繪結構

- **視角構圖**：正面坐姿，雙手自然放於茶器旁，目光前視但略向下，姿態內斂。
- **背景處理**：仿茶室之竹屏、燈籠或書畫掛軸為隱約背景，採淡金褐與墨綠色調，氣氛寧靜。
- **肌理與質感**：布料自然皺摺細膩，面容沉靜但神韻十足，整體呈現侘寂之中蘊藏的精神強度。

## 五、總評

這張茶人型立繪角色成功演繹了戰國時代中以「文化」為政略工具的獨特人物象，如千利休、今井宗久等人。他們既非一線武將，也不以政令見長，卻能透過茶道、藝術與禮法，成為君臣之間、諸侯之間的重要連結與潤滑劑。此角設若實裝於《信長之野望》系列，可擔任提升外交成功率、維穩民心與增強文化建設的關鍵人物，是幕後靜制天下的文化型內政智將。
